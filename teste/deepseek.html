Dicas de Performance Visual

  • ✅ Use opacidade em vez de novas cores
  • ✅ Mantenha apenas 2-3 cores de destaque
  • ✅ Evite sombras pesadas (use 0 1px 3px rgba(0,0,0,0.05))
  • ✅ Dê espaçamento generoso entre elementos
  • ✅ Use tons da mesma família para hierarquia
  • ✅ Contraste moderado, sem forçar a vista
